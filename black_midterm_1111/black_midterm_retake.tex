    Y_n = \frac{2 X_n +1}{X_n + 10}
    \)
    \item {[5]} The average number of snowmen that can be built each day:
    \(
    S_n = \frac{1}{n} \sum_{i=1}^n W_i^2.
    \)
\end{enumerate}
